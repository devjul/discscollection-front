# discscollection-front

Discscollection is a simple SPA in order to work with useEffect, use, <Suspense /> with a GraphQL API

## Installation

This is a React single-page application (SPA) using Vite and React. To run it locally, you'll need Node.js installed. Then, simply run:

```bash
npm install
npm run dev
```

Then open [http://localhost:5173/](http://localhost:5173/).

## GraphQL API

You can install the GraphQL API using this repo [https://github.com/devjul/discscollection-api](https://github.com/devjul/discscollection-api)
